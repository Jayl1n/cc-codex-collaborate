# Contributing

Thanks for contributing to `cc-codex-collaborate`.

Recommended contribution areas:

- better planning-review schemas
- safer sensitive-operation detection
- more robust status/state handling
- language-specific human-question templates
- examples for different tech stacks
- tests for scripts and hooks

Before submitting changes:

1. Validate JSON files with `jq empty`.
2. Run ShellCheck on shell scripts if available.
3. Avoid adding secrets, real credentials, or private project data.
4. Update `CHANGELOG.md` when behavior changes.
